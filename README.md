# 传感器监控系统 (Sensor Monitoring System)

一个基于 React + TypeScript 构建的现代化传感器数据监控与管理系统，支持实时数据采集、可视化分析和样品配置管理。

- **飞人羽翼**

## 项目简介

本系统是一个专业的工业级传感器监控解决方案，用于实时监测和记录温度、重量及视觉检测数据。系统提供直观的用户界面，支持多种传感器设备的并行监控，并具备完善的数据分析和导出功能。

## 核心功能

### 1. 实时监控
- **温度传感器**：通过红外传感器实时监测环境温度
- **重量传感器**：串口天平数据采集，精确到 0.01g
- **视觉检测**：实时视频流显示，支持尺寸测量（长/宽/高）
- **水切宽度计算**：可选的专业测量功能

### 2. 数据管理
- **数据录入表格**：自动记录时间戳、传感器数据和样品信息
- **三种视图模式**：
  - 表格视图：详细数据列表
  - 趋势图：可选择不同指标的时序分析（使用 Recharts）
  - 占比图：重量分布饼图，自动分类统计
- **数据导出**：支持 CSV 和 Excel 格式导出

### 3. 样品配置
- **标准样品**：预置标准样品 A 和 B
- **自定义样品**：用户可添加、查看、删除样品配置
- **参数范围设置**：为每个样品配置温度、重量、尺寸等参数的上下限

### 4. 设备连接
- **串口天平配置**：波特率、数据位、停止位、校验位可调
- **红外传感器配置**：串口参数自定义
- **摄像头管理**：1280x720 分辨率视频流
- **连接状态监控**：实时显示各设备连接状态

### 5. 权限管理
- **登录认证**：管理员登录系统（默认口令：admin123）
- **功能权限**：未登录用户只能查看，登录后可添加/删除样品配置

## 技术栈

### 前端框架
- **React 18.3.1**：现代化 UI 框架
- **TypeScript 5.7.3**：类型安全
- **Vite 6.3.5**：快速开发构建工具

### 样式与组件
- **Tailwind CSS v4.1.12**：原子化 CSS 框架
- **Shadcn UI**：基于 Radix UI 的高质量组件库
- **Lucide React**：现代化图标库

### 路由与状态
- **React Router v7.13.0**：客户端路由
- **React Hooks**：状态管理

### 数据可视化
- **Recharts 2.15.2**：图表库（折线图、饼图）

### 设备通信
- **Web Serial API**：串口设备通信（仅 Chrome/Edge）
- **WebRTC getUserMedia**：摄像头访问

### 通知系统
- **Sonner 2.0.3**：优雅的 Toast 通知

## 浏览器兼容性

- **推荐浏览器**：Chrome 89+, Edge 89+（支持 Web Serial API）
- **其他浏览器**：Firefox, Safari（部分功能受限，串口功能不可用）

## 系统要求

- **Node.js**：v18.0.0 或更高版本
- **包管理器**：npm 或 pnpm
- **操作系统**：Windows / macOS / Linux

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
# 打开浏览器访问 http://localhost:5173
```
